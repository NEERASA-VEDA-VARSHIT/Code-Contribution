# Dart Directory

This directory contains code examples written in Dart. Dart is a client-optimized language for fast apps on any platform.

## Code Examples

### 1. Hello World

The `hello-world.dart` script prints "Hello, World!" to the console.

```dart
void main() {
  print('Hello, World!');
}
```
